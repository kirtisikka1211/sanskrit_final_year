# Devanagari Font

Please download Noto Sans Devanagari font and place it here as `NotoSansDevanagari-Regular.ttf`

## Download Instructions:

1. Visit: https://fonts.google.com/noto/specimen/Noto+Sans+Devanagari
2. Click "Download family" or download the Regular variant
3. Extract the font file
4. Place `NotoSansDevanagari-Regular.ttf` in this directory (`assets/fonts/`)

Alternatively, you can download directly from:
https://github.com/google/fonts/raw/main/ofl/notosansdevanagari/NotoSansDevanagari-Regular.ttf

Save it as `assets/fonts/NotoSansDevanagari-Regular.ttf`
